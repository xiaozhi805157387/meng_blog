android studio 的logcat(日志)筛选栏消失的
发表于 2019-12-18| 更新于 2019-12-18|android
阅读量:

前奏

最近不知道为什么老出现这个情况，很纠结，像我这种要去不同的设备查看信息，遇到这种情况不能筛选整个人都要气猛了
![url]{https://upload-images.jianshu.io/upload_images/5797132-e49738d826cb757e.jpg?imageMogr2/auto-orient/strip|imageView2/2/w/1200}

解决方案

一、重启

把开发软件关闭，然后重新打开就会有了，如果还没有，则重启电脑再看看，缺点太麻烦了

二、位置切换

右键点击logcat头部，选择 MoveTo 任意一个选项，例如left top，然后解决问题之后选择 Bottom Left 或者 Bottom Right。 缺点 展示没发现啥
