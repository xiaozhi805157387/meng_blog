前言

在最近学习 kotlin 的时候，看到每个对象/属性后面都有几个函数

  • let
  • also
  • with
  • run
  • apply
    话不多说直接看兄啊过

1.let 函数

  • 使用方法

2.also函数

其实跟let函数差不多，但区别是返回值吧

  • 使用方法

  • 区别

    • let 是根据最后一行的结果决定res是什么
    • also 是传入对象的本身

3.with 函数

调用同一个对象的多个方法/属性时，去除对象名

  • 使用方法

4.run函数

跟with函数有点类似，不过多一个返回值。可以做判空处理

  • 使用方法

5.apply函数

跟run 函数类似，跟also函数一样，会返回对象本身，不是最后一行
